Autores

Convenience library for working with embedded Java™ resources. Use annotations for verification and code generation.

  • Validate resource paths with the compiler
  • Mitigate against stringly typed things
  • Reduce resource file I/O handling
  • Embed files as byte arrays or strings
  • Replace localization resource bundle messages with typed method signatures

This library can affect compile time, class sizes, and constant pools. Use judiciously.

Sample


      @ClasspathResource(
        value = {
                "Poule.txt",
                "Roses.txt",
        },
        handler = GenerateStringsFromText.class
      )
      public class PrintRhymes {

        public static void main(String...args)  {
          System.out.println(Poule.text());
          System.out.println(Roses.text());
        }
      }
